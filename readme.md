## Markdown ##

Tekst formatteren met *markdown* (md):

``` 
**vet**
_cursief_
```

**vet**
_cursief_

* Lijsten met *, -  aan het begin van de regel
   - onderlijst
   - 2e regel onderlijst
   * met ster
* En de 2e regel.
Zie ook [github markdown guide](https://guides.github.com/features/mastering-markdown/)
